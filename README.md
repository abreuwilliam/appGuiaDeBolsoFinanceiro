Guia de Bolso – Controle Financeiro Pessoal

Aplicativo desenvolvido para ajudar pessoas a organizar suas finanças pessoais de forma simples e visual.

O Guia de Bolso permite registrar gastos, acompanhar saldo disponível, visualizar relatórios financeiros e entender melhor para onde o dinheiro está indo.

O objetivo do projeto é oferecer uma ferramenta prática de controle financeiro, além de servir como estudo de arquitetura backend e desenvolvimento de aplicações completas.
